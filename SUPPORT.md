# Likely Support

Likely is maintained by **The Orange Design**. For help, email [we@theorange.design](mailto:we@theorange.design) or open a GitHub issue at <https://github.com/TheOrangeDesign/likely/issues>.

## Before requesting support

1. Update Likely to the latest version.
2. Confirm the active page begins with `https://www.facebook.com/` or `https://web.facebook.com/`.
3. Reload the Facebook tab after installing or updating the extension.
4. Open the correct post's reaction count and leave the reaction dialog visible.
5. Try a new session with the default settings.

## What to include in a bug report

- Likely version from `chrome://extensions`;
- Chrome version and operating system;
- the exact status or error displayed by Likely;
- the steps immediately before the problem;
- whether the issue happens on every post or one post only;
- a screenshot with names, profile photos, URLs, account details, and other personal information removed.

Do not include passwords, cookies, access tokens, private profile information, or an unredacted reaction list.

## Security and privacy reports

For a suspected security or privacy issue, do not publish sensitive details in a public issue. Open a minimal issue asking the maintainer for a private reporting channel. Do not attach credentials or personal data.

## Scope of support

Support covers installing Likely, its controls, and reproducible extension errors. It cannot restore Facebook accounts, remove Facebook restrictions, alter Facebook limits, or guarantee compatibility after Facebook interface changes.

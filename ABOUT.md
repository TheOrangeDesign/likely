# About Likely

Likely is a focused Chrome extension for administrators of Facebook Pages. After the user opens the reaction list for a specific post, Likely finds visible **Invite** buttons and activates them using the limits and cooldown selected by the user.

Likely does not choose a Facebook account, Page, post, audience, or reaction filter. The user remains responsible for opening the correct Page and post before starting a session.

## What Likely does

- Operates only after the user opens the extension on a supported Facebook tab and presses **Start**.
- Processes the reaction dialog that is already open.
- Tracks successful invitations for the current session.
- Supports pause, resume, stop, session-limit, and cooldown controls.
- Stops when the dialog is closed, the session limit is reached, or no additional Invite buttons are found.

## What Likely does not do

- It does not collect or transmit reaction names, profiles, credentials, cookies, or browsing history.
- It does not run a server, analytics service, advertising system, or remote script.
- It does not sign in, switch profiles, open posts, bypass warnings, solve checkpoints, or guarantee that Facebook will accept an invitation.
- It is not affiliated with, endorsed by, or sponsored by Meta or Facebook.

Likely is open source at <https://github.com/TheOrangeDesign/likely>.

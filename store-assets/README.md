# Store Asset Guide

This folder contains the reusable branded promotional images for a hypothetical Chrome Web Store listing.

## Included

- `small-promo-440x280.png` — required small promotional tile
- `marquee-1400x560.png` — optional marquee promotional image

## Screenshots still required

Capture at least one actual-product screenshot at **1280x800** (preferred) or **640x400**. Up to five may be used.

Recommended sequence:

1. Likely popup with the reaction dialog open and status Ready.
2. Likely running with a small successful invitation count.
3. Likely paused, demonstrating user control.
4. About & Privacy page.

Use only an authorized test Page/post. Before saving a screenshot, blur or replace every person's name, profile image, post URL, account identifier, notification, message preview, and other personal information. Keep screenshots square-cornered, full bleed, current, and free of unsupported claims.

Do not use a mock Facebook screenshot as if it were the actual product experience.

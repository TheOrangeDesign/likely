# Likely Privacy Policy

**Effective date:** July 22, 2026  
**Last updated:** July 22, 2026

Likely is designed to operate locally in the user's Chrome browser. This policy explains the limited information the extension accesses and how it is handled.

## Information Likely accesses

When the user opens Likely on a Facebook post and starts a session, the extension temporarily reads the visible structure and button labels inside the already-open Facebook reaction dialog. This access is necessary to locate visible buttons labeled **Invite**, determine whether a click succeeded, and scroll the dialog.

Likely also stores the user's extension settings—session limit, cooldown frequency, and cooldown duration—using Chrome's local extension storage.

## Information Likely does not collect

Likely does not collect, transmit, sell, share, or retain:

- Facebook usernames, profile names, profile links, photos, reactions, posts, or messages;
- passwords, authentication cookies, access tokens, or other login information;
- general browsing history or activity outside the user-selected Facebook tab;
- analytics, advertising identifiers, location information, or payment information.

## Processing and storage

All webpage processing occurs locally in the browser. Session status exists only in the memory of the Facebook tab and is cleared when the tab is reloaded or closed. User settings remain in Chrome's local extension storage until the extension is removed, its data is cleared, or the settings are replaced.

Likely has no backend server and sends no user data to the developer or third parties. It does not use analytics, advertising, tracking pixels, or remotely hosted executable code.

## Permissions

- **activeTab:** Gives Likely temporary access to the Facebook tab only after the user opens the extension.
- **scripting:** Allows the packaged Likely page script to run in that user-selected tab and interact with the open reaction dialog.
- **storage:** Saves the user's session-limit and cooldown preferences locally.

## Data sharing and sale

Likely does not share or sell user data. Because no user data is transmitted to the developer, developer personnel cannot view it.

## Security

Likely minimizes access by using temporary, user-initiated tab permission. The extension contains its executable code within the installed package and does not download remote code.

## Children

Likely is a utility intended for people authorized to administer a Facebook Page. It is not directed to children and does not knowingly collect personal information from anyone.

## Changes to this policy

Material changes to Likely's data practices will be described in an updated policy and, when required, disclosed to users before the changed practices begin.
